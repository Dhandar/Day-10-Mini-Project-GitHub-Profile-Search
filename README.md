# GitHub Profile Search

## Description

Search any GitHub user and view their profile details.

## Features

- Search GitHub users
- Display profile dynamically
- Last 5 search history
- Loading indicator
- Error handling
- Responsive design

## API Used

https://api.github.com/users/{username}

## How to Run

- Open index.html in browser
- Type Username
- Click search button
- Data show on Screen
